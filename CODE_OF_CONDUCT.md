# 社区行为准则

SoundPort 希望提供一个友善、专业且尊重彼此的开源协作环境。

## 我们的承诺

参与者应当：

- 尊重不同经验、背景、观点和沟通方式
- 针对技术与行为进行讨论，不攻击个人
- 接受建设性反馈，并在产生误解时善意澄清
- 尊重隐私，不公开他人的个人信息或设备标识
- 把社区安全和项目长期健康放在个人胜负之前

以下行为不可接受：

- 骚扰、歧视、威胁或人身攻击
- 性化内容或不受欢迎的关注
- 恶意披露隐私、凭据或安全漏洞细节
- 持续破坏讨论、发布垃圾内容或故意误导

## 执行

维护者可以编辑或删除不当内容，并对违反准则的参与者作出警告、临时限制或永久封禁。
处理事件时会尽量保护报告者隐私，并依据行为的严重程度和持续性采取相称措施。

如果问题涉及安全漏洞，请使用 [SECURITY.md](SECURITY.md) 中的私密报告方式。
其他行为问题可以通过 GitHub 的私密联系方式报告给仓库维护者。
